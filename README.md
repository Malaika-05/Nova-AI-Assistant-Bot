# Nova — AI Study Assistant Bot

Nova is an intelligent learning companion built for students. Powered by Groq's LLM API and Flask, Nova helps you understand topics faster, generate quizzes instantly, solve problems step by step, and memorize with flashcards — all in one place.

## Features
- 💬 **Chat mode** — Ask anything, get instant AI answers
- 📝 **Summarize mode** — Get a full topic summary in seconds
- 🧠 **Quiz mode** — Auto-generates MCQs with scoring
- 🔢 **Solve mode** — Step-by-step solutions for math & logic
- 🃏 **Flashcard mode** — Flip cards to memorize definitions
- 🎙️ **Voice input** — Hands-free questions via Web Speech API

## Tech Stack
- **Backend:** Python, Flask
- **AI:** Groq LLM API
- **Frontend:** HTML, CSS, JavaScript
- **Deployment:** Render
